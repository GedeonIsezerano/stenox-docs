## Bring Your Own Key

Stenox supports Bring Your Own Key (BYOK) — use your personal API keys with cloud providers. This gives you full control over your accounts and often access to free tiers.

<Tip>
  API keys are stored securely in your **macOS Keychain** and never leave your Mac except to authenticate with your provider.
</Tip>

## Transcription Providers

### DeepGram

**Best for:** High accuracy, real-time transcription

<Steps>
  <Step title="Create Account">
    Visit [console.deepgram.com](https://console.deepgram.com) and sign up
  </Step>

  <Step title="Get Free Credits">
    New accounts receive **$200 in free credits** — no credit card required initially
  </Step>

  <Step title="Create API Key">
    Go to **API Keys** → **Create Key** → Copy the key
  </Step>

  <Step title="Add to Stenox">
    Settings → **Providers** tab → **DeepGram** → Paste key → Save
  </Step>
</Steps>

[Learn more about DeepGram →](/providers/transcription/deepgram)

### OpenAI Whisper

**Best for:** Official OpenAI, multi-language

<Steps>
  <Step title="Create Account">
    Visit [platform.openai.com](https://platform.openai.com) and sign up
  </Step>

  <Step title="Add Payment">
    Go to **Settings** → **Billing** → Add payment method
  </Step>

  <Step title="Create API Key">
    Go to **API Keys** → **Create new secret key** → Copy immediately
  </Step>

  <Step title="Add to Stenox">
    Settings → **Providers** tab → **OpenAI** → Paste key → Save
  </Step>
</Steps>

<Warning>
  OpenAI API keys are shown only once. Copy and store securely before closing the dialog.
</Warning>

[Learn more about OpenAI Whisper →](/providers/transcription/openai-whisper)

**Best for:** Free tier, excellent quality

## Local Providers (No Key Required)

### WhisperKit (Transcription)

<Steps>
  <Step title="Open Settings">
    Settings → **Models** tab
  </Step>

  <Step title="Select WhisperKit">
    Set **Transcription Provider** to **WhisperKit**
  </Step>

  <Step title="Download Model">
    Choose a model (base.en recommended) and click **Download**
  </Step>
</Steps>

No API key needed. Runs entirely on your Mac.

[Learn more about WhisperKit →](/providers/transcription/whisperkit-local)

---

### MLX (AI Enhancement)

<Steps>
  <Step title="Open Settings">
    Settings → **Models** tab
  </Step>

  <Step title="Select MLX">
    Set **AI Enhancement Provider** to **MLX**
  </Step>

  <Step title="Download Model">
    Choose a model (Qwen 3 1.5B recommended) and click **Download**
  </Step>
</Steps>

<Warning>
  MLX requires Apple Silicon. Intel Mac users must use cloud providers.
</Warning>

[Learn more about MLX →](/providers/llm/mlx-local)
