## Overview

Stenox Cloud Transcription is our managed cloud service that provides fast, accurate speech-to-text without any API key configuration. It's included with Plus and Lifetime Plus subscriptions.

**Key Benefits:**
- No API keys to manage
- Optimized for Stenox workflows
- Fast, reliable infrastructure
- Included with Plus/Lifetime Plus

## Availability

| Plan | Stenox Cloud |
|------|--------------|
| Free | Not included |
| Plus | Included |
| Lifetime | Not included |
| Lifetime Plus | Included |

<Note>
  Free and Lifetime users can use [local providers](/providers/local-privacy) or [BYOK cloud providers](/providers/cloud-byok) instead.
</Note>

## Setup Instructions

<Steps>
  <Step title="Activate License">
    Ensure your Plus or Lifetime Plus license is activated in Settings → **License**
  </Step>

  <Step title="Select Stenox Cloud">
    Go to Settings → **Providers** tab → Set **Transcription Provider** to **Stenox Cloud**
  </Step>

  <Step title="Start Dictating">
    No API keys needed — just start using Stenox!
  </Step>
</Steps>

## When to Consider Alternatives

- **Maximum privacy:** Use [WhisperKit](/providers/transcription/whisperkit-local) for local processing
- **Specific provider preference:** Use [BYOK](/providers/cloud-byok) with your preferred service
- **Free tier only:** Use [Groq Whisper](/providers/transcription/groq-whisper) or [WhisperKit](/providers/transcription/whisperkit-local)

## Comparison

| Feature | Stenox Cloud | WhisperKit | BYOK Providers |
