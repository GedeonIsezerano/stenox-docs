## Overview

Stenox Cloud AI Enhancement is our managed cloud service that provides high-quality text enhancement without any API key configuration. It's included with Plus and Lifetime Plus subscriptions.

**Key Benefits:**
- No API keys to manage
- Optimized for Stenox workflows
- High-quality grammar and formatting
- Included with Plus/Lifetime Plus

## Availability

| Plan | Stenox Cloud AI |
|------|-----------------|
| Free | Not included |
| Plus | Included |
| Lifetime | Not included |
| Lifetime Plus | Included |

<Note>
  Free and Lifetime users can use [local MLX](/providers/llm/mlx-local) or [BYOK cloud providers](/providers/cloud-byok) instead.
</Note>

## Setup Instructions

<Steps>
  <Step title="Activate License">
    Ensure your Plus or Lifetime Plus license is activated in Settings → **License**
  </Step>

  <Step title="Select Stenox Cloud">
    Go to Settings → **Providers** tab → Set **AI Enhancement Provider** to **Stenox Cloud**
  </Step>

  <Step title="Start Dictating">
    No API keys needed — your transcriptions will be automatically enhanced!
  </Step>
</Steps>

## What Stenox Cloud Enhancement Does

### Grammar & Punctuation
- Fixes grammatical errors
- Adds proper punctuation
- Corrects tense and subject-verb agreement

### Formatting
- Proper capitalization
- Number and date formatting
- Sentence structure improvements

### Customization
- Use custom prompts in Profile settings
- Adjust tone (professional, casual, etc.)
- Add specific formatting instructions

## When to Consider Alternatives

- **Maximum privacy:** Use [MLX](/providers/llm/mlx-local) for local processing
- **Specific model preference:** Use [BYOK](/providers/cloud-byok) with OpenAI GPT-4o
- **Free tier only:** Use [Google Gemini](/providers/llm/google-gemini) or [Groq](/providers/llm/groq-llms)
