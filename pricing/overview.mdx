## Free Tier Details

The free tier includes:

- **5 minutes of dictation per day** (resets at midnight)
- **All features** — profiles, vocabulary, file transcription
- **All providers** — local and BYOK cloud
- **No credit card required**

<Tip>
  **Unlimited free use:** Combine free tier with BYOK providers that have their own free tiers (Groq, Gemini). Your Stenox 5-minute limit only applies to Stenox Cloud.
</Tip>

[Learn more about Free Tier →](/pricing/free-tier)

## Stenox Cloud

Stenox Cloud is our managed service that eliminates API key management:

- **Stenox Cloud Transcription** — Fast, accurate speech-to-text
- **Stenox Cloud AI Enhancement** — High-quality grammar and formatting

Included with **Plus** and **Lifetime Plus** plans.

<Note>
  Stenox Cloud is optional. You can always use local providers (WhisperKit, MLX) or BYOK cloud providers with any plan.
</Note>

## Upgrade Anytime

Start with Free and upgrade when you need more:

1. **Free → Plus**: Monthly subscription for unlimited use + Stenox Cloud
2. **Free → Lifetime**: One-time purchase for unlimited BYOK use
3. **Free → Lifetime Plus**: One-time purchase for unlimited use + Stenox Cloud
4. **Plus → Lifetime Plus**: Switch from subscription to one-time purchase
