## Free Tier Details

The free tier includes:

- **$5 Stenox Cloud credit** to get started
- **5 minutes of dictation per day** (resets at midnight)
- **All features** — profiles, vocabulary, file transcription
- **All providers** — local, BYOK, and Stenox Cloud
- **Upgrade to Lifetime anytime**

[Learn more about Free Tier →](/pricing/free-tier)

## Stenox Cloud

Stenox Cloud is our managed service that eliminates API key management:

- **Stenox Cloud Transcription** — Fast, accurate speech-to-text
- **Stenox Cloud AI Enhancement** — High-quality grammar and formatting

Included with **Plus** and **Lifetime Plus** plans.

<Note>
  Stenox Cloud is optional. You can always use local providers (WhisperKit, MLX) or BYOK cloud providers with any plan.
</Note>

## Upgrade Anytime

Start with Free and upgrade whenever you're ready:

- **Free → Plus**: Monthly subscription with unlimited Stenox Cloud
- **Free → Lifetime**: One-time purchase for unlimited BYOK and local use
- **Free → Lifetime Plus**: One-time purchase with unlimited Stenox Cloud included

No pressure. Your $5 credit and all features work great for getting started.
