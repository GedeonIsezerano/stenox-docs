## What are Profiles?

Profiles let you create different Stenox configurations for different scenarios. Each profile can have:

- Different transcription provider
- Different AI enhancement provider
- Custom enhancement prompts
- Auto-activation rules

<Tip>
  **Example:** Professional email profile with DeepGram + Gemini, and quick notes profile with WhisperKit only.
</Tip>

## Default Profile

Stenox includes a **Default** profile used when no other profile is active.

**Default settings:**
- Transcription: Your last configured provider
- AI Enhancement: Your last configured provider
- Auto-activation: None (manual only)

## Common Profile Setups

### Work Email Profile

**Configuration:**
- **Transcription:** DeepGram Nova-3 (accurate)
- **Enhancement:** Google Gemini 2.5 Flash
- **Custom prompt:** "Professional business tone"
- **Auto-activate:** When Gmail or Outlook is active

### Quick Notes Profile

**Configuration:**
- **Transcription:** Groq Whisper (fast)
- **Enhancement:** None (skip for speed)
- **Auto-activate:** When Notes app is active

### Privacy Profile

**Configuration:**
- **Transcription:** WhisperKit (local)
- **Enhancement:** MLX (local)
- **Auto-activate:** When specific confidential apps are active

### Code Documentation Profile

**Configuration:**
- **Transcription:** DeepGram Nova-3
- **Enhancement:** Gemini 2.5 Flash
- **Custom prompt:** "Technical documentation style, preserve code terms"
- **Auto-activate:** When VSCode or IDE is active

## Profile Features

### Custom Enhancement Prompts

Each profile can have a unique enhancement prompt:

**Professional:**
```
Rewrite in professional, business-appropriate tone.
Use formal language. Format as clear email.
```

**Casual:**
```
Fix grammar but keep casual, friendly tone.
Use contractions. Keep it conversational.
```

**Technical:**
```
Format for technical documentation.
Preserve all code and technical terms exactly.
Use active voice and clear language.
```

### Color Coding

Assign colors to profiles for easy identification in the menu bar.

## Switching Profiles

### Manual Switching

1. Click Stenox menu bar icon
2. Hover over "Profiles"
3. Select desired profile

Active profile shows checkmark.

### Auto-Activation

Profiles can automatically activate based on:
- Active application (app bundle ID)
- Window title (e.g., "Gmail" in browser title)

[Learn about auto-activation →](/profiles/auto-activation)

## Profile Management

**Edit profile:**
- Settings → Profiles tab
- Select profile
- Modify settings
- Save changes

**Delete profile:**
- Select profile
- Click Delete
- Confirm deletion
- (Cannot delete Default profile)

**Duplicate profile:**
- Select existing profile
- Click Duplicate
- Modify as needed
- Save with new name

## Example Profile Strategy

**Beginner (2 profiles):**
1. **Default:** DeepGram + Gemini (cloud, quality)
2. **Offline:** WhisperKit + MLX (local, fallback)

**Intermediate (4 profiles):**
1. **Work Email:** DeepGram + Gemini Professional
2. **Quick Notes:** Groq (no enhancement, fast)
3. **Documentation:** DeepGram + Gemini Technical
4. **Offline:** WhisperKit + MLX (fallback)

**Advanced (6+ profiles):**
- Per-application profiles with auto-activation
- Different languages
- Privacy vs quality trade-offs
- Cost optimization profiles
