## Troubleshooting

Before contacting support, try these common solutions:

<AccordionGroup>
  <Accordion title="No text appearing after dictation" icon="text-slash">
    - Check Accessibility permission: System Settings → Privacy & Security → Accessibility → Stenox enabled
    - Click in a text field before dictating
    - Restart Stenox

    [Full troubleshooting guide →](/guides/troubleshooting#no-text-appears-after-dictation)
  </Accordion>

  <Accordion title="Hotkey not working" icon="keyboard">
    - Check for conflicting apps (Spotlight, Alfred, Raycast)
    - Try a different hotkey in Settings
    - Verify Stenox is running (icon in menu bar)

    [Full troubleshooting guide →](/guides/troubleshooting#hotkey-not-working)
  </Accordion>

  <Accordion title="API key errors" icon="key">
    - Verify the complete key was copied
    - Check key is active in provider dashboard
    - Try creating a new API key

    [Full troubleshooting guide →](/guides/troubleshooting#api-key-errors)
  </Accordion>

  <Accordion title="Slow processing" icon="clock">
    - Use a cloud provider for faster results
    - Try smaller local models (base.en instead of small.en)
    - Skip AI enhancement for fastest output

    [Full troubleshooting guide →](/guides/troubleshooting#slow-processing--long-wait-times)
  </Accordion>
</AccordionGroup>

## Contact Support

If you can't find an answer in the documentation:

### In-App Feedback (Recommended)

The easiest way to contact us:

1. Open Settings → **About** → **Send Feedback**
2. **Dictate your issue** — speak naturally
3. Your message is transcribed and sent automatically

Or use the feedback hotkey: **Right ⌘ + Right ⌥** from anywhere.

<Tip>
  Using the in-app feedback lets you describe your issue naturally with your voice — much easier than typing!
</Tip>

[Learn more about sending feedback →](/support/feedback)

### Email

Email us at [support@stenox.app](mailto:support@stenox.app)

**Please include:**
- Description of the issue
- Steps to reproduce
- macOS version (About This Mac)
- Stenox version (Settings → About)
- Provider you're using

## Response Times

We aim to respond to all support requests within 24-48 hours on business days.

For urgent issues, include "URGENT" in your subject line.
